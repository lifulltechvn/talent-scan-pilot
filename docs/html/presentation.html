  • 3

§2.3 · Data Flow

Pipeline AI thông minh

  • ①
    Extract text (PDF/DOCX/Scanned)
    PyMuPDF + python-docx · OCR: Claude Sonnet Vision (~5s)
  • ②
    PII Filter + Injection Guard
    Regex ẩn danh email/SĐT/CCCD · 11 pattern detect injection
  • ③
    Claude Haiku — parse structured data
    ~3s · ~$0.002/call → JSON: skills, exp, education, language
  • ④
    Titan Embed V2 — generate vector
    ~0.5s · ~$0.00004 → 1024-dim embedding saved to pgvector
  • ⑤
    Smart Pool — auto-match
    pgvector cosine similarity vs all open jobs · combined = cosine 30% + skill 70%

§4 · Thách thức & Giải quyết

Vấn đề thực tế gặp phải

  • 1
    Bedrock throttling khi batch 200 CV
    Root cause: rate limit per-region · Fix: multi-region round-robin + exponential backoff (1s→2s→4s) · Result: 0% failed
  • 2
    CV scan (ảnh) không extract được text
    Root cause: PyMuPDF trả empty · Fix: detect <100 chars → fallback Claude Sonnet Vision OCR · Result: 100% coverage
  • 3
    Prompt injection trong CV
    Root cause: ứng viên chèn "ignore instructions" · Fix: 11 pattern regex + XML delimiter wrapping · Result: blocked 100%
  • 4
    Skill name không chuẩn hóa
    Root cause: "ReactJS" vs "React.js" vs "React" · Fix: skill normalizer mapping + AI-assisted synonym detection

§5 · Security & AI Safety

Bảo mật — Production-ready

🔐JWT + bcrypt — 3 roles (admin/hr/interviewer)✅

⚠️Injection guard — 11 pattern + XML wrapping✅
📋Audit log — mọi action ghi user + entity + time✅
🗑️GDPR erasure — cascade delete all candidate data✅
💰AI cost tracking — mỗi call log model + tokens + cost✅
🔄Retry + fallback — 3 attempts, exponential backoff✅

§8 · Bài học & Tâm đắc

Điều học được

  • 1
    Hybrid scoring > pure AI scoring
    AI hallucinate scores → rule-based làm anchor 70% giúp kết quả ổn định, reproducible
  • 2
    PII protection phải ở layer riêng
    Không tin tưởng prompt instruction "đừng lưu PII" — phải filter TRƯỚC khi gọi AI
  • 3
    pgvector đủ tốt cho scale vừa
    Không cần Pinecone/Qdrant — same DB đơn giản ops, query đủ nhanh ≤1000 vectors
  • 4
    Retry + multi-region = reliability
    Bedrock throttle thường xuyên — round-robin + backoff giải quyết hoàn toàn
  • 5
    Cost tracking là feature, không phải afterthought
    Log mỗi AI call → dashboard chi phí → HR thấy value proposition rõ ràng

Cách hệ thống sử dụng Skill Maps

Khi tạo Job → chọn category → hệ thống load skill map tương ứng → so sánh skills ứng viên (từ AI parse) với required skills của grade level. Scoring engine dùng skill map để tính skill_score component (30% tổng điểm Rule). AI cũng tham chiếu skill map khi đánh giá LLM score.

API Endpoints

← Docs

60+ REST endpoints. Swagger UI tại /docs. Tất cả protected trừ login/register.

🔐 Auth: Bearer JWT token required. Lấy token qua POST /api/auth/login → dùng trong header Authorization.

Auth

POST/api/auth/loginLogin → access + refresh token
POST/api/auth/registerRegister user (admin only)
POST/api/auth/refreshRefresh access token
GET/api/auth/meCurrent user info

Jobs

GET/api/jobsList all jobs
POST/api/jobsCreate job + auto-embed
GET/api/jobs/{id}Job detail + candidates
PUT/api/jobs/{id}Update job
POST/api/jobs/{id}/suggestAI suggest candidates
PUT/api/jobs/{id}/assignAssign candidates
POST/api/jobs/{id}/scoreScore all assigned
POST/api/jobs/{id}/generate-jdAI generate JD

Candidates

GET/api/candidatesList (filter, search, paginate)
GET/api/candidates/{id}Detail + score + matched jobs
PUT/api/candidates/{id}/statusUpdate status
DEL/api/candidates/{id}GDPR erasure (cascade)
GET/api/candidates/{id}/matched-jobsJobs matched to candidate

CV Upload

POST/api/cv-uploadSingle CV upload + parse
POST/api/cv-batchBatch upload (max 200)
GET/api/cv-batch/{id}Batch status
GET/api/cv-batch/{id}/itemsBatch items detail

Scoring

POST/api/scoring/scoreScore single candidate × job
GET/api/scoring/{candidate_id}Get score details

Interviews

GET/api/interviewsCalendar list
POST/api/interviewsSchedule interview
PUT/api/interviews/{id}Update interview
POST/api/interviews/{id}/feedbackSubmit feedback
POST/api/interviews/{id}/questionsAI generate questions

Outreach & Email

POST/api/outreach/sendSend personalized email
GET/api/outreach/templatesList email templates
PUT/api/outreach/templates/{type}Update template

Dashboard

GET/api/dashboard/overviewStats + action items
GET/api/dashboard/ai-costsAI usage & costs

Quiz (Verification)

POST/api/quiz/generateAI generate quiz for candidate
GET/api/quiz/{id}Get quiz + questions
POST/api/quiz/{id}/submitSubmit responses
